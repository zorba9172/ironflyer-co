# Pricing Model

Ironflyer should avoid unlimited plans.

Unlimited AI execution destroys margin.

## Recommended Pricing Structure

### Free Trial

- very small credits
- preview-limited
- no premium reasoning
- strict abuse limits
- watermark or limited export

### Developer

- monthly platform fee
- prepaid execution credits
- standard reasoning
- Vercel previews
- limited storage
- basic SDK

### Team

- higher platform fee
- shared wallet
- better concurrency
- priority providers
- VS Code extension
- OpenVSCode workspace
- more storage

### Business

- higher margin
- SSO
- audit
- team permissions
- higher limits
- support
- private blueprints

### Enterprise

- custom pricing
- dedicated tenant isolation
- custom policies
- private model endpoints
- compliance
- SLA
- dedicated support

## Chargeable Units

- execution credits
- premium reasoning
- sandbox minute
- preview deployment
- mobile build
- storage GB/month
- private model inference
- enterprise governance
- advanced support
