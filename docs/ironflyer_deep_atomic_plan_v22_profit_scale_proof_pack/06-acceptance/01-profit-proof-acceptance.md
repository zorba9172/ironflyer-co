# Profit Proof Acceptance Criteria

## Paid Execution

- user budget required before execution
- wallet/top-up exists
- execution credits deducted
- budget reservation works
- stop-loss works

## Cost Tracking

- provider cost tracked
- sandbox cost tracked
- storage cost tracked
- deployment cost tracked
- support cost approximated
- margin calculated

## Profit Guard

- blocks unprofitable premium calls
- kills idle sandboxes
- degrades under budget pressure
- asks for more budget when ROI is clear
- records decisions

## Scale

- paid queue measured
- worker capacity measured
- event lag measured
- recovery rate measured
- margin under load measured

## Proof

- dashboard shows revenue/cost/margin
- cohort dashboard shows repeat usage
- blueprint dashboard shows cost reduction
- ledger can reconstruct execution economics
