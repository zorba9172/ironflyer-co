# Scenario Model

The system must simulate profitability before scale.

## Inputs

- active users
- executions per user per month
- average user spend per execution
- average platform cost per execution
- completion rate
- refund rate
- repeat usage
- provider cost mix
- sandbox cost
- storage cost
- support cost

## Outputs

- MRR
- provider spend
- infrastructure spend
- gross profit
- gross margin
- completion count
- failed paid execution count
- support burden
- break-even users
