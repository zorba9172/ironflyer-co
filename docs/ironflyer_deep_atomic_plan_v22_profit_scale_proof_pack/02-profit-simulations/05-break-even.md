# Break-Even Model

## Formula

```text
break_even_executions =
fixed_monthly_cost
/
(avg_revenue_per_execution - avg_variable_cost_per_execution)
```

## Example

```text
fixed_monthly_cost = $20,000
avg_revenue_per_execution = $18
avg_variable_cost_per_execution = $6.30

contribution_margin = $11.70

break_even_executions = $20,000 / $11.70 = 1,710 executions/month
```

If average user runs 4 executions/month:

```text
break_even_users = 1,710 / 4 = 428 paying users
```

## Rule

Scale is only good if contribution margin stays positive.
