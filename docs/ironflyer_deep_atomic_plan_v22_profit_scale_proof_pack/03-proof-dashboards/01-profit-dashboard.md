# Profit Dashboard

Simple dashboard metrics:

## Overview

- revenue today
- user credit consumed today
- provider spend today
- sandbox spend today
- gross margin today
- active executions
- blocked executions
- unprofitable executions blocked
- refunds
- top-up rate

## Required Formula Cards

```text
gross_margin_pct
completion_per_dollar
avg_cost_per_execution
avg_revenue_per_execution
break_even_progress
```

## Operator Questions

- Are users paying enough to cover runtime?
- Are we profitable today?
- Which executions are leaking margin?
- Which providers are too expensive?
- Which blueprints are most profitable?
