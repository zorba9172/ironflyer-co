# Blueprint Profit Dashboard

Blueprints are a margin engine.

## Metrics By Blueprint

- executions
- avg revenue
- avg platform cost
- gross margin
- preview success
- completion score
- repair count
- time to preview
- refund rate
- repeat usage after blueprint

## Decision

Promote blueprints that:
- complete faster
- cost less
- produce higher repeat usage
- have lower repair rate

Retire blueprints that leak margin.
