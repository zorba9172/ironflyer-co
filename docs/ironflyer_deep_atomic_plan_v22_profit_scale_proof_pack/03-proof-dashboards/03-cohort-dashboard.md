# Cohort Dashboard

Profit proof requires retention.

## Cohorts

Track users by first paid execution month.

## Metrics

- first paid execution conversion
- second execution rate
- 7-day repeat usage
- 30-day repeat usage
- top-up rate
- refund rate
- support tickets per execution
- average spend per tenant
- gross margin per tenant
- completion rate by cohort

## Healthy Signal

Users return after the first preview.

Unhealthy signal:
Users pay once, fail, and never return.
