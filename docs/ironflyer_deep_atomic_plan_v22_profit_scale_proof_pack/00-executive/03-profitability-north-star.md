# Profitability North Star

Main metric:

```text
Profitable Completed Execution Rate
```

Definition:

```text
profitable_completed_execution_rate =
completed_paid_executions_with_positive_margin
/
total_paid_executions
```

## Supporting Metrics

- gross margin per execution
- gross margin per tenant
- completion-per-dollar
- time to first preview
- cost to first preview
- provider spend per execution
- sandbox cost per execution
- repair cost per execution
- refund rate
- top-up rate
- repeat execution rate
- human support minutes per execution
- stuck execution rate
- zombie kill rate
- recovery success rate
