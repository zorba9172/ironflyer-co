# 90-Day Scale Proof Plan

Goal:
Prove scalable unit economics and runtime stability.

## Build

- 10 blueprints
- persistent runtime v1
- affected verification v1
- private classifiers v1
- provider arbitration v1
- canary brain governance
- mobile approval app
- Figma import v1

## Target

- 150 paying users
- 1,000+ paid executions
- gross margin > 55%
- preview success > 80%
- repeat usage > 35%
- stuck execution rate < 8%
- recovery success > 60%

## Pass Criteria

```text
margin improves as usage grows
```

If cost grows linearly with usage, the model is not working.
