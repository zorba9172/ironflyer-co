How it works

From signature to finality.

Three on-chain steps. No wrapped tokens. No federated bridges. No "trust us" middleware.

01

Lock on source

You sign a transfer on the source chain. The router deposits into the unified liquidity pool and emits a verifiable event with the destination payload.

02

Verify the proof

Decentralized verifiers attest to the source event using Merkle proofs. A k-of-n quorum is required before the message is admitted on the destination.

03

Settle on destination

The router on the destination chain unlocks native liquidity to your wallet. No wrapped IOUs, no synthetic claims. The asset you receive is the canonical one.

Features

Built for actual users.

Six things every bridge claims. We ship them with on-chain proof and audited code.

∞

Unified liquidity

One pool per asset across every chain. No fragmented liquidity, no idle capital, no thin routes during high-volatility windows.

⚡

Sub-minute finality

Median settlement under 30 seconds. Optimistic finality on L2-to-L2 routes; canonical finality on mainnet, with an upper bound published per chain.

⛨

MEV-protected routing

Routes execute through a private mempool with commit-reveal pricing. Front-runners can't sandwich your bridge.

↻

Refundable transfers

If a transfer cannot settle within its timeout, the source contract refunds automatically. No support tickets, no manual recovery.

$

Native asset out

You receive the canonical token on the destination chain — not a wrapped derivative. ETH stays ETH, USDC stays USDC.

◐

Composable messages

Bridge plus call. Trigger a swap, deposit, or contract call on the destination atomically with the bridge transfer.

Security

Developers

Integrate in 10 minutes.

Drop the SDK into your app. Quote, route, and execute with one method. Webhooks for transfer state, queryable proofs for settlement.

bridge.ts 
import { Stargate } from "@stargate/sdk";

const quote = await Stargate.quote({
  from: { chain: "ethereum", token: "USDC" },
  to:   { chain: "base",     token: "USDC" },
  amount: 1_250_000_000n, // 1,250 USDC (6 decimals)
});

// quote.fee = 580_000n (0.05%), quote.eta = 24s

const tx = await Stargate.execute(quote, { signer });
await tx.waitForSettlement();
// ✓ settled at block 21,448,902 · native USDC on Base
