Engine

The gate runtime, visible.

A short sample from a real run: the orchestrator wires gates, the security gate blocks a leaked key, and the deploy gate refuses to ship until the issue is patched.

finisher/run.go 
// gates run in order; each verdict is appended to the ledger.
for _, g := range finisher.DefaultGates() {
    verdict, cost := g.Evaluate(ctx, env)
    ledger.Post(verdict, cost)

    if verdict.Pass {
        continue
    }
    return finisher.Halt(g.Name(), verdict.Reason)
}

// security gate blocked: secrets in env.local
// deploy gate refused: precondition unmet
// run halted at gate=security in 2.3s, cost=$0.014
