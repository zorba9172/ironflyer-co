protocol

How a feed gets signed.

No black-box. The protocol is documented, the contracts are public, and every signed round is reproducible from chain state.

01 / collect

Operators pull data

Each of the 32 operators pulls the raw data from its independent source set — exchanges, RWA custodians, RNG entropy, or off-chain compute results.

02 / aggregate

Median + outlier reject

The off-chain aggregator computes a trimmed median across submissions, rejecting outliers beyond 1.5σ. The resulting value is signed by each operator.

03 / publish

Quorum settles on-chain

A relayer submits the signed payload to the on-chain verifier. The contract checks k-of-n signatures, deviation, and heartbeat, then writes the round.

products

Four products. One network.

The same operator set, the same signing scheme, four very different on-chain capabilities.

$

Price feeds

Sub-cent precision across 214 pairs. Spot, TWAP, and protected reference rates. Used by lending, perps, stablecoins, and treasury accounting.

⚂

Verifiable randomness

VRF backed by drand and threshold BLS. Useful for NFT mints, fair launches, gaming, and committee selection. Bias-resistant, request-response.

λ

Off-chain compute

Run user-defined computations off-chain, post the result with signatures. Replaces gas-heavy on-chain math: matrix ops, ML inference, range proofs.

↔

Cross-chain attestations

Sign state from one chain, verify it on another. Used for cross-chain governance, omnichain accounting, and L2-to-L2 messaging fallback.

security

Slashed for misreporting.

Operators bond ORAC and are slashed for collusion, deviation, or downtime. The signing path is open-source, audited, and verified by a public reference implementation.

Read a feed in 3 lines.

Solidity, Rust, or TypeScript. Reference contracts on every chain, deterministic addresses, and a single CLI to subscribe to any feed.

consumer.rs
pragma solidity ^0.8.24;

import { IOracleFeed } from "@oracle-data/contracts/IOracleFeed.sol";

contract VaultPricer {
    IOracleFeed public immutable ethUsd;

    constructor(address _feed) { ethUsd = IOracleFeed(_feed); }

    function value(uint256 ethAmount) external view returns (uint256) {
        (int256 price, uint256 updatedAt) = ethUsd.latestRoundData();
        require(block.timestamp - updatedAt < 300, "stale price");
        return uint256(price) * ethAmount / 1e8;
    }
}
