§02 — System

The system — bone, ink, blue.

Every brand I ship has a colour discipline, a type discipline, and a behaviour discipline. The bone/ink/blue you see on this site is mine.

01 / Colour

Bone, ink, blue.

Bone for paper. Ink for rules. Blue when it has to bite. Anything else has to argue its way in.

02 / Type

Archivo Black + Mono.

A display face that doesn't blink, a working face that does the dishes, and a monospace for the small print.

03 / Behaviour

Three rules — no more.

Heavy rules, big type, single accent. If a project needs a fourth move, that's the project, not the system.

§03 — Manifesto

EIGHT RULES.

  1. If the logo doesn't work at one centimetre, it doesn't work.

  2. A brief is the client's first draft. Rewrite it.

  3. Big type is honest. Small type is honest, too. The middle is suspicious.

  4. A system survives a junior designer's bad day. A logo doesn't.

  5. One accent colour. Earn the second one.

  6. The launch is week one. The brand is years three to ten.

  7. If a stakeholder won't read the brand book, the brand book is too long.

  8. Charge for the thinking. Give the artwork away with it.
