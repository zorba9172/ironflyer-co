---

# Core Design Principles

### 1. Prompt First

The prompt is the product.

Everything else is secondary.

Layout hierarchy:

1. Headline
2. Prompt Input
3. CTA
4. Templates
5. Features

Nothing should compete with the prompt box.

---

### 2. Soft Cyber

Most AI startups overdo cyber.

Avoid:

* Matrix effects
* Hexagons
* Hacker fonts
* Green terminals
* Overwhelming neon

Use:

* Subtle glows
* Soft gradients
* Atmospheric lighting
* Neural energy

Think:

"Google built a cyberpunk product."

---

### 3. Floating Intelligence

Nothing should feel boxed.

Instead:

* floating panels
* soft elevation
* ambient depth

The interface should feel alive.
