How it works

Pay once. Verified forever.

Files are encoded with Reed-Solomon, replicated across 32 nodes, and verified by hourly proof-of-storage challenges. Lose a node, the network heals automatically.

step 01 · encode

Encode and shard

Your file is Reed-Solomon encoded into 32 shards. Any 21 shards can reconstruct the file. Each shard is hashed and signed before it leaves your browser.

step 02 · replicate

Replicate to nodes

Shards are uploaded to 32 independent storage nodes across three continents. The on-chain registry records the node set, the file hash, and the payment receipt.

step 03 · prove

Prove storage hourly

Each node submits a proof-of-storage every hour. A node that fails to prove loses its bond. The network repairs the file by replicating from another shard set.

Features

What permanent actually means.

No subscriptions. No renewal emails. No "we shut down the project, please come pick up your files." On-chain references that resolve in any client.

∞

Pay once, store forever

A single upfront payment funds the storage endowment for that file. The endowment yield pays node operators continuously.

⛬

32-node replication

Reed-Solomon coding with k=21, n=32. Survives 11 simultaneous node failures with zero data loss.

⌗

Proof of storage

Every hour, every node submits a verifiable proof that it still holds your shard. Failed proofs trigger slashing and automatic re-replication.

$

Pay in ARK or USDC

Price is denominated in USD, paid in USDC or ARK. Stable cost per MB. No "gas spiked and your upload failed" surprises.

↻

HTTP gateway resolvable

Files resolve at gateway.archive.io/<hash>, or directly via the on-chain registry. Compatible with IPFS and Arweave gateways.

⛨

Encrypted by default

End-to-end encryption with libsodium. The network stores ciphertext; only you and the recipients you grant hold the decryption key.

Simple pricing

One payment. Forever.

Cost is a function of file size only. Encoding, replication, proof-of-storage, and bandwidth are all included. No bandwidth tiers, no egress fees, no tiers.

Security

Audited. Bonded. Recoverable.

Every storage node bonds ARK before it can host shards. Misreporting or downtime slashes the bond. The protocol layer is open-source, audited by three firms, and covered by a $1M bug bounty.

Developers

One method. Stored forever.

Drop the SDK into your stack. Upload, get a permanent hash, embed it anywhere a URL goes. Works in Node, browser, and Deno.

upload.ts 
import { Archive } from "@archive/sdk";

const archive = new Archive({ payer: signer, currency: "USDC" });

const result = await archive.upload(file, {
  encrypt: true,
  metadata: { title: "Contributor Agreement", author: "Field & Format" },
});

console.log(result.hash);     // ar://0x4f3b…a92e
console.log(result.cost);     // $0.00184 (paid once)
console.log(result.gateway); // https://gateway.archive.io/0x4f3b…a92e
